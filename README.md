# Brainfuck Compiler

Compiles [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) programs to C.

### Usage:
	> bfc input-file output-file

### Example Programs:
#### Hello World:
	++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.

#### Echo single character:
	,.

#### Print the alphabet:
	++++++++++++++++++++++++++>+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++<[>.+<-]
